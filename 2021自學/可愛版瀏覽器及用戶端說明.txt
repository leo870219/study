瀏覽器先生在這裡人生地不熟，想知道 google 怎麼走，於是跑去問 DNS Server 大大 ： 不好意思，請問 google 怎麼走呀～
DNS Server 大大在這行幹了好幾百年了，這點小事難不倒它，以迅雷不及掩耳的速度，翻開他的資料庫，找到 google 的 IP 位置，並且告知瀏覽器先生。
瀏覽器先生道謝完後，把 IP 位置輸入到全球網域中請求結果（request），這時相對應掌管 google 網頁的 Server 接線小姐收到瀏覽器先生的通知了！
Server小姐 跑去問她家的資料庫大哥瀏覽器先生要找的東西。
資料庫大哥找到後回傳訊息給 Server 小姐。
Server 小姐再把她收到的訊息 response 給瀏覽器先生。
瀏覽器先生拿到訊息後，靠他自己的能力，最後把訊息給具現化（顯示網頁）。
